## Descripción

Aplicación web progresiva para el estudio de la Constitución Española. Incluye flashcards, tests cronometrados, modo examen, sistema de logros con niveles y XP, racha diaria, estadísticas de progreso y repaso inteligente de errores.

## Tecnologías

- HTML5 / CSS3 / JavaScript vanilla
- Service Worker para funcionamiento offline
- localStorage para persistencia de progreso
- Diseño responsive

## Características principales

- 10 títulos constitucionales con artículos completos
- Flashcards interactivas para estudio
- Tests cronometrados con diferentes modos
- Sistema de gamificación: niveles, XP, logros y racha diaria
- Repaso inteligente de preguntas falladas
- Estadísticas detalladas de progreso
- Instalable como PWA
