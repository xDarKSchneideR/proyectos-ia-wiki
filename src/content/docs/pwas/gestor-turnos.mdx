## Descripción

Calendario mensual para la gestión de turnos de trabajo. Permite registrar lugar, hora de entrada y salida, descanso en minutos, notas y tipo de turno con colores personalizables. Calcula automáticamente horas netas trabajadas, pago estimado según tarifa horaria y genera resúmenes semanales.

## Tecnologías

- HTML5 / CSS3 / JavaScript vanilla
- Tailwind CSS (via CDN)
- localStorage para persistencia
- Exportación CSV y JSON

## Características principales

- Calendario mensual interactivo
- Registro de turnos con hora, lugar y notas
- Colores personalizables por tipo de turno
- Cálculo automático de horas netas y pago estimado
- Resumen semanal de horas trabajadas
- Exportación e importación de datos
- Impresión optimizada
