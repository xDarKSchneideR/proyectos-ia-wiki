## Descripción

**Web:** [xdarkschneider.github.io/GarantiaBox](https://xdarkschneider.github.io/GarantiaBox/)

Gestor 100% offline para registrar productos con garantía. Permite añadir productos con fecha de compra y período de garantía, adjuntar fotos de tickets (comprimidas en cliente), recibir alertas visuales de caducidad con código de colores, realizar backup y restore en JSON, y buscar o filtrar por fecha o estado. Protegido con PIN de 4 dígitos.

## Tecnologías

- HTML5 / CSS3 / JavaScript vanilla
- Service Worker para offline-first
- Compresión de imágenes en cliente
- localStorage para persistencia
- Web Share API
- Modo oscuro automático

## Características principales

- Registro de productos con fecha y período de garantía
- Captura y compresión de fotos de tickets
- Alertas de caducidad por colores (verde, amarillo, rojo)
- Backup y restore completo en JSON
- Búsqueda y filtros por fecha y estado
- Protección con PIN de 4 dígitos
- Instalable como PWA
