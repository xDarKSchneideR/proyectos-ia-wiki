## Descripción

Aplicación profesional para medir áreas y perímetros sobre mapas interactivos. Permite dibujar polígonos, medir distancias, crear puntos de interés agrupables por colores, añadir notas con texto e imágenes, y exportar datos. Con autenticación de usuarios y persistencia en PostgreSQL.

## Tecnologías

### Frontend

- React 19 + Vite 6 + TypeScript
- Tailwind CSS 4
- Leaflet + React-Leaflet + Turf.js
- MarkerCluster para agrupación de marcadores
- PWA con vite-plugin-pwa

### Backend

- Express + Node.js + TypeScript
- PostgreSQL (Neon)
- Autenticación JWT + bcrypt

## Características principales

- Dibujo de polígonos sobre mapas interactivos
- Medición de distancias y áreas precisas
- Puntos de interés con agrupación por colores
- Notas con texto e imágenes
- Exportación de datos en múltiples formatos
- Autenticación segura de usuarios
- Persistencia en base de datos PostgreSQL
