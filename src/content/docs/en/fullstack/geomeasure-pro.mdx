## Description

Professional application for measuring areas and perimeters on interactive maps. Draw polygons, measure distances, create color-coded points of interest, add notes with text and images, and export data. Includes user authentication and PostgreSQL persistence.

**Website:** [geomeasurepro.netlify.app](https://geomeasurepro.netlify.app/)

## Technologies

### Frontend

- React 19 + Vite 6 + TypeScript
- Tailwind CSS 4
- Leaflet + React-Leaflet + Turf.js
- MarkerCluster for point grouping
- PWA with vite-plugin-pwa

### Backend

- Express + Node.js + TypeScript
- PostgreSQL (Neon)
- JWT + bcrypt authentication

## Key features

- Polygon drawing on interactive maps
- Precise distance and area measurement
- Color-coded points of interest
- Notes with text and images
- Multi-format data export
- Secure user authentication
- PostgreSQL database persistence
