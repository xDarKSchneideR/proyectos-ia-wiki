## Description

Progressive web app for studying the Spanish Constitution. Includes flashcards, timed tests, exam mode, an achievement system with levels and XP, daily streak, progress statistics, and smart error review.

**Website:** [constitucion-pro.vercel.app](https://constitucion-pro.vercel.app/)

## Technologies

- HTML5 / CSS3 / JavaScript vanilla
- Service Worker for offline support
- localStorage for progress persistence
- Responsive design

## Key features

- 10 constitutional titles with full articles
- Interactive flashcards for study
- Timed tests with different modes
- Gamification: levels, XP, achievements, daily streak
- Smart review of wrong answers
- Detailed progress statistics
- Installable as PWA
