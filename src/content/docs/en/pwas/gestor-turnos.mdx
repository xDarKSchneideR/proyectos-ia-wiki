## Description

Monthly calendar for managing work shifts. Log location, clock-in and clock-out times, break duration, notes, and shift type with customizable colors. Automatically calculates net hours worked, estimated pay based on hourly rate, and generates weekly summaries.

## Technologies

- HTML5 / CSS3 / JavaScript vanilla
- Tailwind CSS (via CDN)
- localStorage for persistence
- CSV and JSON export

## Key features

- Interactive monthly calendar
- Shift logging with time, location, and notes
- Customizable colors per shift type
- Automatic net hours and pay calculation
- Weekly work hours summary
- Data export and import
- Print-optimized layout
