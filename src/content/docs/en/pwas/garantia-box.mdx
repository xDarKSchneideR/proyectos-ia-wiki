## Description

100% offline warranty manager for products. Add products with purchase date and warranty period, attach ticket photos (client-side compression), receive expiry alerts with color coding, backup and restore via JSON, and search or filter by date or status. Protected with a 4-digit PIN.

## Technologies

- HTML5 / CSS3 / JavaScript vanilla
- Service Worker for offline-first
- Client-side image compression
- localStorage for persistence
- Web Share API
- Automatic dark mode

## Key features

- Product registration with date and warranty period
- Ticket photo capture and compression
- Expiry alerts with color coding (green, yellow, red)
- Full backup and restore via JSON
- Search and filter by date and status
- 4-digit PIN protection
- Installable as PWA
